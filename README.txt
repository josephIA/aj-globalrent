AJ GlobalRent & Services - Static MVP (GitHub + Netlify ready)
Folder: aj-globalrent/

Quick start:
1. Create a GitHub repo named 'aj-globalrent' under your account (josephIA).
2. Upload all files from this folder to the repo root.
3. Go to Netlify, click 'Add new site' -> 'Import from Git' and connect to your repo.
4. In Netlify, enable Identity and Git Gateway for Netlify CMS (see Netlify docs).
5. Visit https://<your-site>.netlify.app/admin to log in and start adding listings.
